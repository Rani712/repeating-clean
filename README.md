# Repeating Image Transition - Replica

This project is a React + Vite + Tailwind CSS replica of the Codrops "Repeating Image Transition" demo.
It includes:
- Loader state with progress
- Repeating strip transition animation
- Prev/Next controls and dot navigation
- End screen after completing the slideshow

## Run locally

1. npm install
2. npm run dev
3. Open http://localhost:5173

## Deploy
Push to GitHub and connect to Vercel. Vercel will auto-deploy the project.
